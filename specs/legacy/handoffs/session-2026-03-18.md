## Troubleshooting Notes

- `uv run temporal` 返回 exit code 1 并不总是启动失败。
  也可能是人工中断，或远端 SSH 连接失败导致。
- QML 运行时错误应先查看终端栈追踪，
  然后再用 `uv run pyside6-qmllint` 验证。
- 进行 QML 视觉调优时，优先使用相对布局参数，避免在所有区域里
  只靠硬编码像素值调整。
