## Troubleshooting Notes

- `uv run temporal` with exit code 1 is not always startup failure.
  It can be caused by manual interrupt or failed remote SSH connection.
- QML runtime errors should be checked from terminal stack trace first,
  then validated with `uv run pyside6-qmllint`.
- For QML visual tuning, prefer relative layout parameters and avoid
  hardcoded pixel-only adjustments across all sections.
